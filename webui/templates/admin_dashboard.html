Advanced LLM Integration

The system now supports multiple AI models and LLMs for ensemble analysis and richer chat responses. You can configure model selection in sentinal/ai_base.py.
Current Models:

  • Sentiment: distilbert-base-uncased, roberta-base
  • LLM: distilgpt2, gpt2
How it works: Sentiment analysis uses an ensemble (majority vote, average score). Chat responses show outputs from all configured LLMs.
To add models: Edit AIBase in sentinal/ai_base.py and update requirements.txt as needed.
