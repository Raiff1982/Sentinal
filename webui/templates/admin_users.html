User Management Docs

Users are currently managed in-memory in webui/auth.py for demo purposes.
To persist users, integrate a database and update registration/login logic.
Roles: Each user has a role ('admin' or 'user') for feature access.
To add/remove users: Edit the users dictionary in auth.py or implement database-backed CRUD operations.
